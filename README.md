## 🤔 為什麼做這個？

帶看時客戶最常問的問題是：「那我大概要準備多少錢？」

這個工具可以**即時試算不同開價下的自備款金額**，幫助客戶快速了解資金需求，讓議價過程更有依據。

---

## 📋 功能說明

| 功能 | 說明 |
|------|------|
| 基本資訊 | 物件名稱、客戶姓名、看房日期 |
| 坪數計算 | 自動計算（權狀坪數－車位坪數）或手動輸入 |
| 貸款試算 | 按成數或按金額兩種模式 |
| 試算表 | 開價往下每 10 萬一格，自動更新 |
| 費用說明 | 買方應負擔費用一覽（依台灣房屋官方文件） |

---

## 🧮 計算公式

| 項目 | 計算方式 |
|------|---------|
| 房屋實價 | 開價 − 車位價格 |
| 房屋坪數 | 權狀坪數 − 車位坪數 |
| 單價（萬/坪） | 房屋實價 ÷ 房屋坪數 |
| 銀行貸款額 | 銀行預估價格 × 貸款成數 |
| **純自備款** | **開價 − 銀行貸款額** |

---

## 💡 買方應負擔費用（依台灣房屋費用一覽表）

> ⚠️ 以下費用**不包含在試算表內**，帶看時需口頭另行說明

### 📝 簽約時
| 費用項目 | 金額說明 |
|---------|---------|
| 簽約費 | 1,000 元 |
| 仲介服務費 | 簽約同時支付 |

### 📄 簽約後（預收規費）
| 費用項目 | 金額說明 |
|---------|---------|
| 印花稅 | 依稅捐稽關核定 |
| 登記規費、書狀費 | 依地政機關核定 |
| 火險、地震險 | 有向金融機構貸款才有，依保險公司費率計算 |
| 所有權移轉代辦費 | 土地建物買一筆一棟 **14,000 元**起；每增加一地號或建號加收 1,000 元 |
| 貸款抵押設定代書費 | **每件 4,000 元**；每增加一人或一筆地號、建號各加 1,000 元 |
| 實價登錄費 | **每件 1,500 元** |

### 🏠 完稅時
| 費用項目 | 金額說明 |
|---------|---------|
| 契稅 | 依稅捐稽關核定為準 |

### 🔑 交屋時
| 費用項目 | 金額說明 |
|---------|---------|
| 地價稅、房屋稅、水電瓦斯費等 | 交屋日後由買方依比例負擔 |
| 履約保證費 | 按實際**成交價格萬分之3** |

## ⚠️ 免責聲明

本工具為快速試算輔助，**所有數字均為估算**，不代表任何買賣合約之依據。

實際自備款、貸款成數、銀行估價、各項費用金額，須依：
- 個案情況與銀行審核結果
- 地政機關核定費用
- 稅捐稽關實際核定

買賣雙方應負擔費用，請以台灣房屋正式費用說明文件為準，並建議與專業人員確認後再做財務規劃。

**詳請以實際為準。**
